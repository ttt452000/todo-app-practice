#  todo app

day6
chatgptにcodeを出してもらって作った写しです。
HTML / CSS / JavaScript で作ったシンプルな Todo アプリです。  
JavaScript の基礎（DOM操作・イベント・localStorage）の練習として作成しました。

---

##  機能

- タスクの追加
- タスクの削除
- チェックを入れると完了（取り消し線）
- Enterキーでタスク追加
- 完了したタスクを非表示にできる
- localStorage を使ってタスクを保存（リロードしても消えない）

---

##   使用技術

- HTML
- CSS
- JavaScript（Vanilla JS）
- localStorage
- GitHub Actions（学習中）

---

##  学んだこと

- HTML / CSS / JavaScript を使って、はじめてToDoアプリを完成させた
- DOM操作（要素の取得・追加・イベント処理）を実際に手を動かして体験した
- エラー（null / not defined など）に何度も詰まりながら、原因を調べて修正する流れを学んだ
- ChatGPTを活用して実装し、分からない部分はこれから復習予定
